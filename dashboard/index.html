HIL Architecture

Implementation

Production-Grade PID

The PID includes features often missing from textbook implementations: derivative on measurement, low-pass filter, and back-calculation anti-windup.

Controllers.h — PIDController::compute() 
float compute(float sp, float pv) {
  float err = sp - pv;

  // P term
  _pTerm = _Kp * err;

  // D term — derivative on MEASUREMENT (no kick on SP step)
  // Low-pass filter: Tf = Kd/(Kp*N), avoids high-freq amplification
  float dPV = pv - _prevPV;
  float Tf  = _Kd / (_Kp * _N);
  _derivFilter = (1.0f - _Ts/Tf) * _derivFilter - (_Kd/Tf) * dPV;
  _dTerm = _derivFilter;
  _prevPV = pv;

  // Raw output before integral
  float rawOut = _pTerm + _integral + _dTerm;
  float clamped = constrain(rawOut, _outMin, _outMax);

  // Anti-windup: back-calculation — unwinds integrator on saturation
  float satErr = clamped - rawOut;
  _integral += (_Ki * err + _Kb * satErr) * _Ts;

  return clamped;
}
Development

Implementation Roadmap

Phase-by-phase from breadboard to industrial-grade HIL testbed.

Phase 01
Foundation
Hardware Setup & UART Echo
  • Wire ESP32 ↔ Arduino UART at 115200 baud
  • Implement simple echo test firmware
  • Validate byte-level communication
  • Add CRC-16 packet framing
Phase 02
Plant
Plant Simulator Firmware
  • Implement First-Order plant with Euler integration
  • Add 1kHz hardware timer ISR
  • Transmit simulated PV over UART every tick
  • Upgrade to RK4 for Second-Order model
Phase 03
Control
Controller Firmware + Servo
  • Implement On/Off → P → PI → PID progressively
  • Drive servo from control output (0–100% → 0°–180°)
  • Add serial command parser for runtime tuning
  • Implement closed-loop feedback with plant simulator
Phase 04
Advanced
All Plant Models + Fault Injection
  • Add Servo, Tank, DC Motor plant models
  • Implement 7-type fault injection engine
  • Add MetricsEngine (rise time, IAE, overshoot...)
  • Validate each fault type against expected behavior
Phase 05
Dashboard
Python Dashboard + Test Suite
  • Flask + SocketIO real-time data server
  • Four synchronized Chart.js live plots
  • PID tuning UI with live apply
  • 28-test pytest suite + benchmark table
Phase 06
Portfolio
Documentation + Portfolio Polish
  • Write architecture diagrams, theory docs, tuning guide
  • Generate benchmark tables and performance report
  • Record demo video showing all controllers
  • Publish GitHub repository with this showcase page
